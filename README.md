# Smart Methods Web Task

## Project Description

This project was developed using HTML, CSS, JavaScript, PHP, and MySQL.

The website allows users to:
- Enter a name and age.
- Store data in a MySQL database.
- Display all records in a table.
- Toggle the status value between 0 and 1.
- Update the status immediately without reloading the page.

## Technologies Used

- HTML
- CSS
- JavaScript (AJAX)
- PHP
- MySQL

## Project Files

### db.php
Database connection file.
Contains the connection settings between PHP and the MySQL database.

### index.php
Main webpage.
Contains:
- Student Form
- Data insertion
- Display records in a table
- Toggle button.

### toggle.php
Handles status switching.
Changes status values between:
- 0 → 1
- 1 → 0
